# U.S. Vehicle Recall Data Analysis

## Overview
This project analyzes over 300,000 U.S. vehicle recall records to identify trends in recall frequency, severity, and affected components.

## Objective
To explore patterns in recall activity and understand relationships between recall size, severity, and vehicle components.

## Tools Used
- SQL
- Python
- R
- AWS (S3, Glue DataBrew)

## Data Source
National Highway Traffic Safety Administration (NHTSA) public recall dataset.

## Files Included
- Analysis notebook/scripts
- Final report
